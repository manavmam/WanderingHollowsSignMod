**WanderingHollowsSignMod**

A Minecraft mod that introduces custom decorative signs, functional dice mechanics, and new music discs to the game.

**Tech Stack**

Language: Java

Mod Loader and Version: Forge 1.20.1

Build System: Gradle

**Features**

Custom Signs

Functional Dice

Music Discs

**Project Structure**

src/main/java: Contains the logic for item registration and dice functionality.

src/main/resources: Holds all 3D models (.json), textures (.png), and sounds.

build.gradle: Manages dependencies and the compilation process.
